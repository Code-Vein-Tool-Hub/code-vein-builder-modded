Have you found a bug or mistake on Code Vein Builder or just want to submit general feedback? If so, you can let us know by opening a new issue! Please read the information provided below to learn how to submit your report or feedback.

Reporting a Technical Bug

Is Code Vein Builder not working properly on your system? If so, please open a new issue on github and provide the following information.

  • Description of the bug
  • Steps to replicate the bug (if possible)
  • Browser(s) the bug occurred in
  • Operating System
  • Were you using the Online or Offline version?

Note: If you are able to fix the bug you may submit the fix as a pull request instead.

We want to make sure that Code Vein Builder functions properly for everyone, so we greatly appreciate your report and will try to resolve your issue as soon as humanly possible!

Reporting a Mistake

Have you noticed a mistake on Code Vein Builder, such as a spelling error or erroneous information? If so, you can let us know by opening a new issue on github and providing the following information.

  • Erroneous content
  • Correction of the content

Note: If you like, you can fix the mistake yourself and submit the correction as a pull request instead.

We want to make sure that all information is accurate and easy to read, so we greatly appreciate your help in ensuring that!

General Feedback

Does your query not meet the conditions listed above? No problem, we'd still love to hear from you! If you have a suggestion, complaint, or comment, feel free to open a new issue on github to let us know.

  • https://github.com/SethClydesdale/code-vein-builder/issues

Any and all feedback is appreciated; we look forward to hearing from you!
