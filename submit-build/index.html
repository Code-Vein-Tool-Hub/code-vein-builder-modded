Submit a Build

Would you like to submit your build to Code Vein Builder and share it with others? If so, please read the information provided below to learn how to submit your build!

Note: We utilize Github's "Issues" tracker for submitting builds. If you're not familiar with this system you can read about it here. A Github account is required to submit new issues. If you don't have one you can register an account here.

Submitting Your Build

To submit your build to Code Vein Builder please open a new issue on github and provide the following information.

  • Title: Build Submission (feel free to include your build's name in the title)
  • Build Name: Name of Your Build
  • Build Category: (One-handed Sword, Dark Gifts, etc.; See preset list for categories) If you're unsure, you may leave this out or propose a new category.
  • Build Link/Code: (you can get a link/code for your build by setting it up here and selecting "Save Build")
  • Build Description: Describe how to use your build. (optional)

You may copy and use either template:

Plain Text

Markdown (Github)

Here is an example build:

  • Title: Build Submission: Vampiric Mage
  • Build Name: Vampiric Mage
  • Build Category: Dark Gifts
  • Build Link/Code: IIJgHAPsBsEAwQCIEYoBZ5VcSxkggGFltlZQNgBmAVihAE4oqnCQDDWRUAxOIA

Once your build has been submitted on Github, you'll have to wait for approval. If approved, your build should be added within 24hrs! However, if there's any issues, we'll let you know and work with you to get your build added.

Thank you for taking the time to read these guidelines. We look forward to seeing all of your builds!
